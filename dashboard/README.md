# ClaudePlus Hub — Dashboard (read-only)

Веб-панель для наблюдения за общим состоянием хаба: проекты, задачи (очередь и
хэндофф между агентами), общая память (с поиском), история сессий, агенты,
бенчмарки.

**Только чтение.** База открывается в режиме `readonly` — панель физически не
может изменить общее состояние, на которое полагаются агенты (атомарный захват
задач, FTS-триггеры памяти). Запись, если понадобится, должна идти через
MCP-инструменты хаба, а не отсюда.

## Запуск

```powershell
npm run dashboard
# → http://127.0.0.1:4747
```

Слушает только loopback (`127.0.0.1`). Настройки через переменные окружения:

- `DASH_PORT` — порт (по умолчанию `4747`).
- `DB_PATH` — путь к базе (по умолчанию `../data/hub.db` относительно этой папки,
  то есть та же база, что у хаба).

## Перевод на русский (опционально)

Кнопка **🌐 RU** в шапке включает машинный перевод содержимого (описания,
сводки сессий, тела памяти и т.п.) на русский. Подписи интерфейса и так
русские — переводится только то, что агенты записали по-английски.

⚠️ **Приватность.** Перевод выполняется через DeepSeek API, то есть выбранный
текст хаба **отправляется на внешний сервер** `api.deepseek.com` (тем же ключом,
что использует `dsc`). Это происходит **только при включённой** кнопке 🌐 RU;
когда выключено — никаких внешних запросов. Ключ берётся из `DEEPSEEK_API_KEY`
или `~/.deepseek-code/settings.json` и **остаётся на сервере** (в браузер не
передаётся). Каждый уникальный текст переводится один раз и кешируется в
`dashboard/.cache/` (в git не попадает), поэтому повторные просмотры бесплатны.
Тела памяти переводятся лениво — только когда раскрываешь «показать содержимое».

Альтернатива без этого канала: перевод страницы средствами браузера
(правый клик → «Перевести»), но он так же отправляет текст наружу (в Google).

## Как устроено

- `server.mjs` — крошечный Express-сервер: read-only JSON API (`/api/*`) поверх
  `hub.db` через `better-sqlite3`, плюс отдача статики. Никаких новых
  зависимостей — переиспользует `express` и `better-sqlite3` самого хаба.
- `index.html` — одностраничный UI (vanilla JS), стиль «командного центра».
- `vendor/tailwind.js` — Tailwind, заверстан локально (без внешнего CDN):
  работает офлайн, нет риска компрометации CDN.

Стек и идея UI заимствованы из тестового проекта `NexusForge`; сам NexusForge не
изменялся.
